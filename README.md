# 🔐 Malicious URL Detection using Machine Learning

A Machine Learning-based web application that detects whether a URL is **malicious or safe** using advanced feature engineering and ensemble models.

---

## 🚀 Live Demo
👉 http://192.168.1.8:8501

---

## 📌 Overview

Malicious URLs are a major cybersecurity threat used in phishing, malware distribution, and fraud attacks. This project leverages **machine learning models and feature engineering** to classify URLs as **benign or malicious** in real-time.

Machine learning helps overcome limitations of traditional blacklist methods by detecting **previously unseen threats** using patterns in URLs :contentReference[oaicite:0]{index=0}.

---

## ✨ Features

- 🔍 Real-time URL classification
- ⚡ Fast and lightweight prediction system
- 🧠 Ensemble ML model (Logistic Regression, Random Forest, XGBoost)
- 📊 Feature engineering with 20+ URL-based attributes
- 🌐 Deployed using **Streamlit web app**
- 🖥️ User-friendly interface for quick testing

---

## 🧠 Model Details

- **Algorithms Used:**
  - Logistic Regression
  - Random Forest
  - XGBoost (Boosting)

- **Technique:**
  - Stacking Ensemble Model

- **Performance:**
  - ✅ Accuracy: ~95%+
  - ✅ High precision for malicious detection

---

## 📊 Feature Engineering

Extracted key features from URLs such as:
- URL length
- Presence of IP address
- Special characters (@, -, //)
- Suspicious keywords
- Directory depth

These features help identify patterns commonly found in malicious URLs.
